Complete Setup Guide Included

Step-by-step instructions for every configuration. No guesswork.

Total setup time: ~30 minutes
1

Clone Repository

Download the source code from GitHub

git clone https://github.com/Wittlesus/launchfast-starter.git
Time: ~1 minute
2

Install Dependencies

Install all required packages with npm

npm install
Time: ~2 minutes
3

Configure API Keys

Add your environment variables — detailed guide included in README

Database URL: Copy from your PostgreSQL provider (Supabase, Railway, or local)
Stripe Keys: Get from Stripe Dashboard → Developers → API Keys
OAuth Credentials: Create apps in Google Cloud Console & GitHub Settings
Claude API Key: Optional — from console.anthropic.com
Time: ~15 minutes (with step-by-step links in README)
4

Run Database Migrations

Set up your database schema with one command

npx prisma migrate dev
Time: ~1 minute
5

Start Development Server

Launch your app locally and start building

npm run dev
✓ Your app is live at localhost:3000
6

Deploy to Production (Optional)

Deploy to Vercel with one click — free hosting for hobby projects

Time: ~10 minutes (Vercel connects to GitHub automatically)
