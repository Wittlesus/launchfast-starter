Do I need to know Next.js?

Basic React and Next.js knowledge is helpful. The codebase is well-documented with comments explaining key decisions.

What if I don't know Prisma?

Prisma is beginner-friendly. The schema is already set up for users, subscriptions, and payments. Queries look like await db.user.findUnique({ where: { email } }) â€” much easier than raw SQL. The official Prisma docs are excellent if you need to learn more.

Can I use MySQL instead of PostgreSQL?

Yes. Prisma supports PostgreSQL, MySQL, SQLite, SQL Server, MongoDB, and CockroachDB. Just change the provider in schema.prisma from postgresql to mysql and update your DATABASE_URL. The code stays the same.

Do I need a paid Anthropic API key?

Only if you want to use the Claude AI integration. The Anthropic API uses pay-as-you-go pricing (no subscription). You get $5 free credits when you sign up. If you don't need AI features, you can skip the Claude integration entirely â€” the rest of the boilerplate (auth, payments, database) works independently.
